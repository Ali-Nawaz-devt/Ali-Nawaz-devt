## 👨‍💻 About Me

CS student at **Shaikh Ayaz University, Shikarpur** — building ML systems, NLP pipelines, and full-stack applications.

My focus right now is **low-resource NLP** — specifically building AI tools for Sindhi, a language spoken by 30 million people but almost invisible in the NLP world.

- 🔬 Published research: first complete Sindhi sentiment analysis system — **94.8% accuracy**, live on HuggingFace
- 🏆 1st Place — TechnoFun 2024 Programming Competition

## 📝 Research

**Sindhi Sentiment Analysis Using Classical Machine Learning: A Semi-Supervised NLP Pipeline for a Low-Resource Language**
*Ali Nawaz · Shaikh Ayaz University · 2026*

> First complete sentiment analysis system for Sindhi. Three-stage semi-supervised pipeline expanding 848 hand-labeled sentences to 1,909. LinearSVC achieves 94.73% accuracy and 0.993 macro-AUC.

📄 [Dataset on Kaggle](https://kaggle.com/datasets/alinawaz06/sindhi-sentiment-analysis-dataset) · 🤗 [Live Demo](https://huggingface.co/spaces/alinawazmahar/sindhi-sentiment)
